# Horoz İmparatorluğu (Saf JavaScript)

Bu sürüm tamamen tarayıcıda çalışacak şekilde saf HTML/CSS/JavaScript olarak hazırlanmıştır.

## Çalıştırma

1. Proje klasöründeki `index.html` dosyasını çift tıklayın.
2. Oyun doğrudan tarayıcıda açılacaktır.

> Not: Herhangi bir Node.js kurulumu veya derleme adımı gerektirmez.
